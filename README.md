# FackLocation (高级虚拟定位)

一个基于 Xposed 框架的 Android 应用，用于实现系统级的虚拟定位。它可以拦截应用的定位请求，并返回一个预设的地理位置，适用于需要模拟位置的各种场景。

## 功能特性

- **底层 Hook**: 通过 Xposed 直接修改 `android.location.Location` 类的行为，效果稳定。
- **屏蔽辅助定位**: 拦截并屏蔽 WiFi 和基站扫描结果，防止辅助定位暴露真实位置。
- **位置记录**: 可保存多个位置点，方便快速切换。
- **简单易用**: 提供图形化界面进行位置记录和模拟启停。

## 前提条件

1.  **已 Root 的 Android 设备**: 本应用需要 `su` 权限将配置文件写入系统目录。
2.  **LSPosed 或兼容的 Xposed 框架**: 需要通过 Xposed 框架来加载模块并 Hook 系统 API。

## 使用说明

1.  **安装与启用**:
    *   在本项目的 Release 页面下载并安装最新的 APK 文件。
    *   打开 LSPosed 管理器，在“模块”部分找到“高级虚拟定位”并启用它。
    *   在模块的作用域中，勾选你希望应用虚拟定位的目标 App。
    *   重启目标 App 或设备使模块生效。

2.  **记录位置**:
    *   打开“高级虚拟定位”App。
    *   点击右上角的 `+` 按钮，应用会自动获取当前真实的 GPS 位置并弹出保存对话框。
    *   为该位置输入一个名称（如“公司”、“家里”），然后点击“确认”保存。

3.  **开始/停止模拟**:
    *   在 App 主界面的列表中，选择一个你想要模拟的位置。
    *   点击底部的 **“开启模拟”** 按钮。此时，所选位置信息会写入配置文件，Xposed 模块将开始对目标 App 生效。
    *   如需停止，点击 **“停止模拟”** 按钮即可。

## 配置文件

-   应用会将当前选中的模拟位置信息以 JSON 格式写入到设备的 `/data/local/tmp/mock_location.loc` 文件中。
-   Xposed 模块会读取此文件来获取经纬度。
-   该文件权限为 `644`，所有应用均可读取，保证了 Hook 的有效性。
